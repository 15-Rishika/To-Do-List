**To-Do List** application is a Python program built using the pandas and NumPy libraries to manage daily tasks efficiently. It allows users to add, view, remove, and mark tasks as completed through an interactive menu-driven interface. Tasks are stored in a pandas DataFrame with their corresponding status, making the data organized and easy to manipulate. The program also provides basic statistics, showing the total number of tasks, how many are completed, and how many are still pending. It demonstrates key programming concepts such as functions, loops, user input handling, and data manipulation using pandas.
